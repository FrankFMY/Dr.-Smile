# Dr. Smile

Landing page for a dental clinic. Trustworthy medical design with teal palette, doctor profiles, before/after gallery, patient reviews, and online booking form.

## Tech Stack

- **Next.js 16** — App Router, Turbopack
- **Tailwind CSS v4** — `@theme` directive for custom design tokens
- **Framer Motion** — Scroll-triggered animations
- **Lucide React** — Icon system

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
app/components/
  Header.tsx        — Fixed navbar with mobile hamburger menu
  Hero.tsx          — Hero section with clinic introduction
  Services.tsx      — Dental services overview
  Doctors.tsx       — Doctor profiles
  BeforeAfter.tsx   — Treatment before/after gallery
  Reviews.tsx       — Patient testimonials
  Booking.tsx       — Appointment booking form
  Footer.tsx        — Site footer with navigation
```
